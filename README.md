# PrevixOS
PrevixOS is an experimental initiative to redefine how we interact with mobile devices. We believe that modern Android environments have become cluttered and inefficient. Our goal is to strip away the noise and focus on pure, logic-based productivity.

### Status
- This project currently running on Android 16 (Baklava). For more information visit our website - [previxproject.com](https://previxproject.com/).
- [Manifest](https://previxproject.com/manifest/)

### Current Focus
- **Multi Faces Launcher:** Adopting as many as possible different UI from other user interfaces.
- **Desktop Environment:** Using external monitor to use Desktop Mode over USB-2 and USB-3 Protocol.
- **NIPA Charging Protocol:** Charging your device as fast as possible with temperature control.
- **Overclocking Control Panel:** Settings that extend your system restrictions on performance.
